## 💼 주요 프로젝트 및 성과

### **Façade cleaning robot project** | 2023.03 - 2024.12

- **성과**:
  - 비정형 형상 건물(*고척스카이돔)에 **적용 가능한 로봇 시스템을 개발**하여 기존 방식 대비 **작업 영역을 40% 확장**
  - 개발한 로봇 플랫폼으로 **고척 스카이돔 현장 테스트에서 95% 성공률** 달성
  - **2023, 2024 로보월드 전시회**에서 기술 시연 및 업계 관심 유치
- **기술적 성과**:
  - 로프와 Wheel-Leg 메커니즘을 융합하여 **고경사 장애물(45° 이상) 극복**
  - GPS 및 센서 기반 위치추정알고리즘으로 **실시간 로봇 위치 추적 및 제어 정확도 90% 이상** 확보
- **개발 시스템**:
  - **DAR (Dual Ascending Robot)**: 곡면 환경에서 로프등강 이동 플랫폼
  - **RDWR (Rope Driven Wheel-Leg Robot)**: 4족 휠레그 구조로 외벽 장애물 극복 능력 향상
  - **RMA (Rope-Riding Mobile Anchor)**: 작업 영역 확장용 이동식 앵커 포인트

- **성과**:
  - **50.5kg 하중**을 지지하면서도 안정적 로프 주행 구현
  - **49.5kN의 외력**에도 위치 유지 가능한 고성능 앵커 메커니즘 개발

  - MCU와 CAN 네트워크 기반 센서 통합으로 **정밀한 위치 추정 및 제어 구현**

### **DAR (Dual Ascending Robot) 잔류 진동 제어** | 2024.01 - 2024.12
- **성과**:
  - 하드웨어의 잔류 진동 제어 목적의 **RVCA(Residual Vibration Control Algorithm)** 개발
  - 알고리즘 적용 결과 잔류진동 **최대진폭 40.29% 감소, 정정시간 54.02% 단축**
- **기술적 성과**:
  - 로프를 Spring-Damper로 모델링한 **시스템Identification 및 동역학 분석** 수행
  - Input Shaping 제어와 Disturbance Observer를 결합한 **RVCA 제어 알고리즘** 개발
  - **ROS2와 RTOS 통합 프로그래밍**으로 실시간 진동 제어 구현
  - **SCI 저널 논문 투고 (Sensors)**
